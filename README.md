Multiflex Core
==============

Multiflex Core is a fork/derivative of Bitcoin Core. It implements a full node and wallet functionality and can optionally be built with a graphical user interface (Qt).

Project links:

- Website: https://multiflexcoin.com/
- Telegram: https://t.me/multiflexcoin
- Contact: hello@multiflexcoin.com

This repository is an initial, history-free import of the source snapshot that was provided as **Multiflex-Release-1.0.10**.
Snapshot commit reference (from the original archive): `d0f6d9953a15d7c7111d46dcb76ab2bb18e5dee3`.

Upstream project: Bitcoin Core (MIT licensed): https://bitcoincore.org

Build
-----

Build instructions are available in the `doc/` folder (for example the platform-specific build guides).
If you are unfamiliar with building Bitcoin Core-derived projects, start with the relevant `doc/build-*.md` file.
