# mflex_levels icon set

This directory contains Multiflex-specific Qt UI icons (raster PNGs) used to represent "levels" in the application.

## Files

- `miner_level_XX.png` and `miner_level_XX@2x.png`: level icons at standard and high-DPI resolutions
- `source_grid.png`: design/source helper grid used to align/compose the icons
